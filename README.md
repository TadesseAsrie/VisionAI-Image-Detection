# Image Detection System

A modern AI-powered Image Detection System built using HTML, CSS, JavaScript, TensorFlow.js, and COCO-SSD.

## Features

- Upload Images
- Drag & Drop Support
- AI Object Detection
- Bounding Boxes
- Confidence Scores
- Detection History
- Local Storage Support
- Statistics Dashboard
- Export Results
- Dark/Light Mode
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- TensorFlow.js
- COCO-SSD Model

## Installation

1. Download the project.
2. Open the project folder.
3. Open index.html in your browser.

No backend setup required.



## AI Model

This project uses:

- TensorFlow.js
- COCO-SSD Object Detection Model

## Supported Formats

- JPG
- JPEG
- PNG
- WEBP
